# Gauche-text-relative-date

intendet to use parse date text from command-line

- relative-date->date
- date->relative-date

This package roughly computed as Year (= 365 days) Month (= 30 days) .

## Examples (relative-date->date):

Can read followings:

- a day ago
- 2 days ago
- an hour ago
- 2 hours ago
- an hour later

## Examples (date->relative-date)


- just now
- 1 day ago
- 2 days later
- 1 year later
